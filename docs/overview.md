# OpenPencil Overview

Open-source, AI-native design editor. Think Figma, but you can self-host it, extend it, and talk to it.

## Key Features
- **Skia-based Rendering**: High-performance vector rendering using CanvasKit (Skia WASM).
- **AI-Native**: 26+ built-in tools for AI-assisted design.
- **Figma Compatibility**: Full Kiwi encoder/decoder for .fig files.
- **P2P Collaboration**: Real-time sync using WebRTC (Trystero) and Yjs CRDT.
- **Multi-platform**: Tauri for Desktop, Vite for Web, and now Capacitor for Mobile (iPad).
